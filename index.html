Core Calculation Formulas
Basic Salary Cost Core
Basic Salary Cost = Avg Basic Salary × HC
⚠️ May onwards: uses post-adjustment salary
Source: HC Collection + Salary Upload
Overtime Cost Core
OT Cost = Overtime Hours × OT Hourly Rate
OT Rate = Avg Basic Salary ÷ 21.75 ÷ 8 × OT Multiplier
Source: HC Collection (Overtime) + Premise (OT Multiplier)
Social Insurance (SI) Core
SI Cost = Avg SI Base × SI Rate × HC
⚠️ Jul onwards: uses post-adjustment SI base
Source: SI Base Upload + Premise (SI Rate) + HC Collection
PHF Core
PHF Cost = Avg PHF Base × PHF Rate × HC
⚠️ Jul onwards: uses post-adjustment PHF base
Source: PHF Base Upload + Premise (PHF Rate) + HC Collection
Custom & Derived Formulas
Bonus Provision Custom
Bonus = Basic Salary Cost × Bonus Rate ÷ 12
Source: Derived from Basic Salary Cost + Premise (Bonus Rate)
Training Cost Custom
Training Cost = HC × Training Per Head ÷ 12
Source: HC Collection + Premise (Training Per Head)
Total Labor Cost Derived
Total Labor Cost = Basic Salary + OT Cost + SI + PHF + Bonus + Training + [...custom items]
Auto-sum of all active cost components · Adding new formulas will automatically include them
